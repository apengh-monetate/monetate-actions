Install The Monetate Tag

The Monetate tag is a small chunk of code that allows you to test, personalize, merchandise, and measure anything on your website.


There are two different versions of the Monetate Tag:

Asynchronous Tag Synchronous Tag
Non-blocking tag Blocking tag
Monetate fully functional, with the exception of Full-Page testing experiences Monetate fully functional
Single script request Two script requests (entry.js and custom.js)
Loads independent of your site Guarantees Monetate tag load priority
Visual artifacts are possible on your site Prevents visual artifacts from occurring
Can use Content Masking, but more performant on the Synchronous tag Use Content Masking for "Edit" and "Hide" actions. For more information, check out the Content Masking documentation.
13ms response time (average) 11ms response time (average)
