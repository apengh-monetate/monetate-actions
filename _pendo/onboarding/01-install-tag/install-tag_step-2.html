Install The Monetate Tag

Paste ONE of the Monetate Tags below into the first line of the HTML page <head> on every page of your site.


Synchronous Tag:

1
<!-- Begin Monetate ExpressTag Sync v8.1. Place at start of document head. DO NOT ALTER. -->
2
<script type="text/javascript">var monetateT = new Date().getTime();</script>
3
<script type="text/javascript" src="//se.monetate.net/js/2/tag/entry.js"></script>
4
<!-- End Monetate tag. -->
